System Settings

Configuration for the Google Apps Script backend connection.

The endpoint URL is configured in js/config.js. Copy js/config.example.js to js/config.js and paste your Apps Script deployment URL.

How to Update the Backend

  1. Open your Google Sheet
  2. Go to Extensions → Apps Script
  3. Replace the contents of Code.gs with the latest from apps-script/Code.gs
  4. Click Deploy → New Deployment (or Manage Deployments to update existing)
  5. Choose Web App, set Execute as: Me, Access: Anyone
  6. Copy the new URL and update js/config.js
