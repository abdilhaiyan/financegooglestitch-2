Troubleshooting: POST/PUT Failing?

  1. Open Google Sheet → Extensions → Apps Script
  2. Click Deploy → Manage Deployments
  3. Click the pencil (✎) icon on your Web App deployment
  4. Change Version to "New" and click Deploy
  5. Authorize when prompted — this is the key step!
  6. Wait 30 seconds, then click Test Connection above
  7. If still failing, try Deploy → New Deployment and update config.js with the new URL

How to Update the Backend

  1. Open your Google Sheet
  2. Go to Extensions → Apps Script
  3. Replace Code.gs with the latest from apps-script/Code.gs
  4. Deploy → Manage Deployments → Edit (✎) → New Version → Deploy
